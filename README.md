# Catálogo de Filmes

Projeto final da disciplina **Linguagem de Programação para Internet**, desenvolvido utilizando a stack **React + Node.js + Express + MongoDB (via Mongoose)**.  
O sistema tem como objetivo permitir o **cadastro, listagem, edição e exclusão de filmes**, aplicando conceitos de CRUD e integração entre front-end e back-end.

---

## Funcionalidades
- Listar filmes com título, gênero, ano e classificação  
- Cadastrar, editar e excluir filmes  
- Buscar filmes por nome ou gênero  
- Interface web desenvolvida em React  

---

## Tecnologias utilizadas
- **Front-end:** React  
- **Back-end:** Node.js + Express  
- **Banco de dados:** MongoDB (Mongoose)  
- **Controle de versão:** Git e GitHub  

---

## Objetivo
Desenvolver uma aplicação web completa, praticando integração entre front-end e back-end, manipulação de dados com MongoDB e boas práticas de desenvolvimento moderno.
